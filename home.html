Problems with the Website
Usability
Even though the main page looks simplistic. The navigation is super problematic on this website. When the user clicks on Place Order, the page navigates to a site where the user has to find another button to actually see the steps to order food from Plant City. Reserve table button leads to a fairly understandable page in which the user can make a reservation. The overall problem with this user experience is that we don’t know what is on the website at first glance. So the user has to play with the website to understand what the website encapsulates.
Learnability
The website is hard to learn since the user has to play with the website before understanding the functionality of each button. For instance, “Place Order” button does not take you immediately to the page where you can order food on the spot. It redirects the user to another page. In addition, all the information on the home page is not initially displayed. So in order to understand other features of the website excluding ordering food and reserving a table, the user needs to investigate and scroll down.
Memorability
This website is not memorable since it is not easy to navigate and understand. As a first hand user, every time I order from this website I need to relearn the pages since there are a lot of redirects within the website. Although the colors and aesthetic are pleasing. It is not enough to make this page memorable since not everything is displayed such that the user can easily find what they are looking for.
Accesability Issues
From an accessibility point of view, i have found that the navigation system has no errors however there are a lot of redirects on the website that makes certain pages within the website less accessible to the user. From the WAVE report I also saw that the website has very low contrast which makes the site less memorable and harder to navigate. There are also some redundant alternative text and skipped heading levels that makes the site less appealing and more complicated than desired.
